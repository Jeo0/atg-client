# atg-client
automatic trajectory generator cross-platform client for acoustic volumetric displays (dis our thesis :))

# how to build in arch-linux
precheck:
- opengl version must be 4.6. To check, install ```mesa-utils``` 

1. install glfw with
```sudo pacman -S glfw```

2. execute gnu make after navigating in the root folder of the software 
```
cd ./software/;
make
make run
```
